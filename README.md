# portfolio-website

This is a personal portfolio website developed using **HTML, CSS, and JavaScript** to show skills, projects, and contact information

## Features

- Responsive UI
- Smooth scrolling navigation
- Animated background
- Interactive project cards
- Contact section with E-mail, GitHub, LinkedIn
- Feedback form using Formspree

##  Structure

- `index.html` → Main structure of the website
- `profile.jpg` → Profile image

##  HTML Tags & Their Usage

### `<html>`
Root element that contains the entire webpage.

###  `<head>`
Contains metadata, title, and internal CSS styling.

###  `<title>`
Defines the title shown in the browser tab.

###  `<style>`
Used to add internal CSS for styling the webpage.

### `<body>`
Contains all visible content of the website.

###  `<nav>`
Creates the navigation bar.

#### Attributes:
- `href="#section"` → Links to sections within the page

---

###  `<section>`
Defines different sections like About, Projects, Contact.

#### Attributes:
- `id` → Used for navigation and linking

---

###  `<div>`
Used as a container to group elements.

#### Attributes:
- `class` → Used for styling with CSS

---

###  `<img>`
Displays images.

#### Attributes:
- `src` → Path of the image
- `alt` → Alternative text

---

### `<h1>, <h2>, <h3>`
Used for headings and titles.

---

###  `<p>`
Defines paragraphs.

---

###  `<a>`
Creates hyperlinks.

#### Attributes:
- `href` → Link URL
- `target="_blank"` → Opens link in new tab

---

###  `<button>`
Creates clickable buttons.

#### Used with:
- `onclick` → Executes JavaScript functions

---

### `<form>`
Creates a form for user input.

#### Attributes:
- `action` → URL where form data is sent
- `method="POST"` → Sends data securely

---

###  `<input>`
Used for user input fields.

#### Attributes:
- `type="text/email"` → Defines input type
- `name` → Used to identify form data

---

###  `<select>` and `<option>`
Used to create dropdown menus.

---

###  `<textarea>`
Allows multi-line text input.

---

###  `<footer>`
Defines the bottom section of the website.

---
